# Project - graphql-integration
This repository contains integration flows demonstrating integration with GraphQL APIs and mapping approaches

# Steps
1. Download the zip file
2. Open your trial integration tenant
3. Upload the package
4. Deploy the 3 integration flows
5. Ensure you have client id and client secret of your process integration runtime.
6. Get the enpoint URL for the API - `SAP BTP GraphQL API Integration`
7. Test using REST API client with a request PAYLOAD
   ```JSON
   {
    "countryCode": "CA"
   }
   ```
8. You should see the valid response.
